The description of the modules of this folder are

avgCentFull: centered model of the gamma_t profile in the mock framework.
avgMiscFull: mis-centered gamma_t with integration over the \lambda_c, Rmis and \theta
avgMiscApprox1: drop integration over \lambda_c
avgMiscApprox1: drop integration over \lambda_c, Rmis and \theta,
                uses an interoplation table for delta sigma.
