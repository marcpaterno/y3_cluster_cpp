This is a "mock data vector" for the compton_y_sims module. The cosmosis .ini
files which were used to generate it are in this directory. The mass and
redshift bins used for this data vector, as well as the error bars, are
identical to the simulation-derived data vector in `../sz_sims`. This is
intended as a test of the MCMC chain and likelihood, to ensure that we can
recover the correct "known" profile parameters.
